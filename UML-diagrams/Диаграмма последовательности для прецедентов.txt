@startuml
actor Student as s

participant "Приложение" as App
participant "Сервер" as Server
participant "Робот" as Robot
participant "Камера" as Camera

== Вызов робота ==
s -> App: Открыть приложение
s -> App: Запустить вызов робота
App -> Server: Отправить команду на вызов робота
Server -> Camera: Сделать снимок положения робота
Camera --> Server: Вернуть изображение
Server -> Robot: Определить местоположение робота
Server -> Robot: Отправить команду: подъехать к студенту
Robot -> s: Подъехать к столу

== Выбор напитка ==
s -> App: Выбрать напиток
App -> Server: Отправить заказ напитка
Server -> Robot: Отправить заказ на приготовление
Robot -> Server: Подтвердить выполнение команды

== Приготовление напитка ==
Robot -> Robot: Приготовить выбранный напиток
Robot -> Robot: Определить положение стола
Robot -> Server: Сообщить статус готовности
Server -> App: Оповестить студента о готовности
App -> s: Отобразить уведомление

== Подача напитка ==
Robot -> Camera: Захватить изображение стола
Camera --> Robot: Вернуть координаты
Robot -> s: Подать напиток
Robot -> Server: Отметить статус заказа как "выполнен"
Server -> App: Обновить статус заказа
App -> s: Сообщить студенту о завершении

@enduml
